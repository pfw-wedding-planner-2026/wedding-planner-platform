# Wedding Planner Platform

PFW-Projekt von Mohammed Osman und Rihab El Machti.

## Projektidee

Wir entwickeln eine Web-Anwendung zur Planung einer Hochzeit.

Die Anwendung soll Paaren helfen, wichtige Bereiche wie Gäste, Budget, Aufgaben und ausgewählte Services an einem Ort zu organisieren.

## Technischer Schwerpunkt

Das zentrale Feature unseres Projekts ist ein interaktiver Sitzplan- und Platzordnungsplaner.

Nutzer sollen Gäste verwalten und diese später auf Tische und Sitzplätze verteilen können.

Der Sitzplaner soll unter anderem zeigen:

* freie und belegte Plätze,
* maximale Anzahl von Personen pro Tisch,
* noch nicht platzierte Gäste,
* mögliche Konflikte oder Warnungen,
* eine visuelle Übersicht über den Sitzplan.

## Projektumfang

Der Sitzplaner ist das wichtigste Feature.

Andere Bereiche werden bewusst kleiner gehalten:

* Dashboard
* Gästeliste
* Budgetübersicht
* einfache Service-Beispieldaten
* Timeline oder Checkliste

Dadurch bleibt das Projekt in vier Wochen realistisch umsetzbar.

## Geplante Technologien

* Frontend: React mit Vite
* Styling: Tailwind CSS
* Animationen: Framer Motion
* State Management: Zustand oder Redux
* Backend: Spring Boot mit REST API
* Datenbank: PostgreSQL
* Diagramme: Recharts
* Tests: Vitest und React Testing Library
