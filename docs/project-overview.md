# Wedding Planner Platform

## Team
- Mohammed Osman
- Rihab El Machti

## Projektziel
Wir entwickeln eine Web-Anwendung zur Planung einer Hochzeit.

## Technischer Schwerpunkt
Das zentrale Feature ist ein interaktiver Sitzplan- und Platzordnungsplaner.

Nutzer sollen Gäste verwalten und diese auf Tische bzw. Sitzplätze verteilen können.
Die Anwendung soll dabei freie Plätze, belegte Plätze und mögliche Konflikte anzeigen.

## Weitere kleinere Bereiche
- Dashboard
- Gästeliste
- Budgetübersicht
- einfache Service-Beispieldaten
- Timeline oder Checkliste

## Wichtig für unseren Umfang
Der Sitzplaner ist das Hauptfeature.
Andere Bereiche werden bewusst kleiner gehalten, damit das Projekt in vier Wochen realistisch umsetzbar bleibt.
